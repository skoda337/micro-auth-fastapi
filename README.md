# Access Control System

Микросервис на FastAPI для управления пользователями и ролями.

## Функционал
- **SQLAlchemy ORM**: работа с базой данных SQLite.
- **Many-to-Many**: реализована связь между пользователями и ролями.
- **Docker**: проект полностью контейнеризирован.
- **Swagger UI**: интерактивная документация доступна по адресу `/docs`.

## Запуск
1. Соберите образ: `docker build -t micro-auth .`
2. Запустите: `docker run -p 8000:8000 micro-auth`
